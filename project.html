Literary Analysis Engine Project

Author: Elliott Stevanus

Project Mentor: NA

Weekly Meeting Time: 6 hours per week — Wednesdays (6–9 PM) and Fridays (12–3 PM)

Project Summary

This project uses XML to create a website capable of detecting and analyzing metaphors and similes in The Picture of Dorian Gray by Oscar Wilde.

The goal is to compile figurative language results and compare similarities between them — identifying recurring words, shared imagery, and patterns. This allows for clearer visualization of Wilde’s image system and stylistic tendencies.

Ambitious Version

The most ambitious version of this project envisions a highly interactive website. Users would be able to:

  • Click on any highlighted word within a metaphor or simile
  • See how frequently it appears in other figurative constructions
  • Compare usage across metaphors, similes, and other textual contexts
  • Navigate dynamically through the text based on recurring imagery

This would allow readers to explore Wilde’s symbolic and imagistic structure as a connected system rather than isolated examples.
